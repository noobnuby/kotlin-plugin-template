# Kotlin Plugin Template

코틀린 PaperMC Plugin 템플릿입니다.

# Library

* [Run Task](https://github.com/jpenilla/run-task)

# Configuration

gradle.properties

```gradle
group=com.noobnuby.plugin (group)
version=1.0.0 (ver)
paper_version=1.21.8 (papermc ver)
```

# Build

* `./gradlew paperJar`
* `Gradle -> Tasks -> other -> paperJar`

# Run Server

* `./gradlew runServer`
